Задача 4: typeof null
javascript
console.log(typeof null); // "object"
Почему: Историческая ошибка JavaScript с 1995 года. Null должен быть примитивом null, но из-за бага в ранней реализации возвращает "object". Это единственный случай, когда typeof врёт.

Задача 5: "15" + 1
javascript
console.log("15" + 1); // "151"
Почему: Оператор + при строке слева выполняет конкатенацию. Число 1 преобразуется в строку "1", результат — "15" + "1" = "151".

Задача 6: "12" - 1
javascript
console.log("12" - 1); // 11
Почему: Оператор - всегда пытается выполнить вычитание. Строка "12" преобразуется в число 12, затем 12 - 1 = 11 (число).

Задача 7: Boolean("")
javascript
console.log(Boolean("")); // false
Почему: Пустая строка — falsy значение. Boolean() возвращает false для "", 0, null, undefined, NaN, false.
​

Задача 8: Boolean(" ")
javascript
console.log(Boolean(" ")); // true
Почему: Строка с пробелом имеет длину > 0, поэтому truthy. Boolean() проверяет только наличие символов, не их видимость.
